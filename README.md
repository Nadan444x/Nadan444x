### About

AI engineer at **[Margadeshaka](mailto:rohit@margadeshaka.com)** — an AI company building tools for guidance and learning. Day to day I work across the stack: LLMs, full-stack web, MCP servers, and the glue code that makes AI feel native to a product instead of bolted on.

I like building systems where the operator is often an AI rather than a human — issue trackers driven by tool calls, internal apps with chat-first interfaces, MCP servers that turn APIs into things Claude can just *do*.

**Education**: BCA, Panjab University
